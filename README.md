## Overview
This project analyzes publicly available SAT score data to explore relationships between **student performance and school-level factors** such as school size and demographics. The goal was to practice real-world **exploratory data analysis (EDA)** and communicate insights using Python and data visualization.

---

## Objectives
- Clean and prepare public SAT datasets for analysis
- Explore correlations between SAT scores and school characteristics
- Identify trends and patterns in standardized test performance
- Create clear visualizations to communicate findings
